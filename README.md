# IKKOKU — design-vintage-sundries-kottou

> **"Handed down from Edo to Heisei."**

A design study exploring a fictional Kyoto kobijutsu (fine antiques) dealer — third-generation, Kyoto Bijutsu Club affiliated — targeting **a 58-year-old art-history lecturer who collects Edo-period imari, Tang painting, and tea ware** and refuses anything with restoration ambiguity.

IKKOKU is a fictional dealer created for this design study.

## Overview

| | |
|---|---|
| **Brand** | IKKOKU (古美術 一徳) |
| **Persona** | kottou (Antiques / kobijutsu) |
| **Live Site (planned)** | `design.vintage-sundries-kottou.riumu.net` |

## Design Concept

- **Color**: Lacquer black `#0c0a08` × deep navy `#1f2e4a` × old gold `#b88a3a` × porcelain `#f4f0e6` × cinnabar seal `#a8302a`
- **Typography**: Yuji Syuku (display, vertical) × Shippori Mincho B1 (subhead) × Noto Serif JP (body)
- **Aesthetic**: Museum-grade gravity × scroll/hanging-axis layout × vertical-rl heavily used
- **UX**: Tokonoma-style vertical-rl hero, four select works with full metadata, owner essay with red-seal sign-off, reservation-only contact
- **Texture**: Lacquer micro-grain (SVG turbulence with overlay)

## Tech Stack

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN
- No framework, no build step — GitHub Pages ready
